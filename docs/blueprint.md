# **App Name**: LexiLearn AI

## Core Features:

- Word List Generation: Generate a list of 30 spelling words tailored to the user's grade level and past performance, leveraging the Gemini API. The Gemini API will function as a tool using information from user's history stored in local storage, including correct and incorrect words, progress metrics, areas of improvement, usage dates, typing speed, and other stats, with a cutoff of 300 words. This app feature also allows manual control of settings such as difficulty, vocab type.
- Spelling Test: Presents the generated words to the user for spelling practice. Employs a free word-to-speech API for auditory learning. Then will also show sentences and how to use words.
- Local Storage Management: Saves user spelling progress, including correct and incorrect words, test dates, areas for improvement, and typing speed, within the browser's local storage. Manage cap on total size.
- Progress Tracking & Reporting: Analyzes user performance and provides insights into areas of improvement, including detailed stats, typing speed, and tracking all usage.

## Style Guidelines:

- Primary color: A soft blue (#64B5F6) to inspire trust and focus.
- Background color: Light gray (#F0F4F8) provides a neutral, non-distracting backdrop.
- Accent color: A cheerful yellow (#FFCA28) highlights important elements like buttons and correct answers.
- Headline font: 'Space Grotesk' (sans-serif) for a clear, tech-forward feel in titles.
- Body font: 'Inter' (sans-serif) for readability and a clean appearance in longer texts.
- Use friendly, educational icons to represent different grades and achievement levels.
- Maintain a clean, distraction-free layout that will highlight most important components.
